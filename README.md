# valuable-n8n-ai
This repository comprises of all n8n workflows, automations, and AI agents I made with real-life and business-oriented value. It includes AI agents and workflows from "ATS Resume scanners" and "automatic urgent email detection" to "increasing website SEO" and "YouTube video analysis".

# How to use
1. Download the corresponding JSON file of your chosen automation or agent
2. Head over to your n8n and create a new workflow
3. On the top right of the screen, click on the three dots > Import from File
4. Select the downloaded JSON file
5. Follow the configuration guide and steps outlined inside the workflow so it works for you
